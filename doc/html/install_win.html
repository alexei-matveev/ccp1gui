Installation of the software packages for the CCP1GUI under Windows

Base Packages

The easiest way to run the CCP1GUI under windows is to use the Python distribution from Enthought. This is free to students and those in academic institutions.

This provides all the components needed by the CCP1GUI, so if you have this installed you can just download the latest version of the CCP1GUI from sourceforge, unzip it and then run it by double-clicking on the ccp1gui.py file.

If you do not wish to use Enthought, or it is not free for you, then Windows binaries for Python can be downloaded from the Python website

Once you have Python installed, binaries of VTK, numpy, Scientific and Numeric python can be downloaded from those kindly made available on Christoph Gohlke's website.

If using Christoph's binaries, extract the VTK zip file and then copy the folder in there called vtk into:

C:\Python26\Lib\site-packages

(or similar for the version of Python you are using). You will also need to add the full path to the folder:

C:\Python26\Lib\site-packages\vtk

to your PATH environment variable.

The packages for Numeric and/or Scientific are self-installing, so just double-click on the .exe file

You can now just download the latest version of the CCP1GUI from sourceforge, unzip it and then run it by double-clicking on the ccp1gui.py file.

Information on installing the additional packages individually follows below.

Additional Packages

Numerical Python

If you've installed the Enthought package as described above, you already have Numeric installed. If you need to install it separately, please vist the Numerical Python website at sourceforge and click on the Download Numerical Python link. Then select the numpy package to download and you will be presented with a page that offers both installers for windows as well as source builds.

Scientific Python

If you've installed the Enthought package as described above, you already have Scientific Python installed. If you need to install it separately, please vist the Scientific Python website at http://starship.python.net/~hinsen/ScientificPython/. The main page offers both installers for windows as well as downloads of the Python source code.

Python MegaWidgets

Pmw is just pure Python code, and so doesn't need to be installed as such, just copied into a place where Python can find it. A link to a tarred and zipped archive is available on the Pmw webpage. You should be able to unpack the archive with a zip utility such as Winzip, and it will unpack to create a folder called Pmw.

To install Pmw, copy this folder into the Python<VERSION>\Lib\site-packages folder of your Python installation. For example if you are using Python 2.3 and it is installed on the default place on your C: drive, you would copy the Pmw folder to:

C:\Python26\Lib\site-packages

VTK

Binaries of VTK, numpy, Scientific and Numeric python can be downloaded from those kindly made available on Christoph Gohlke's website.

If using Christoph's binaries, extract the VTK zip file and then copy the folder in there called vtk into:

C:\Python26\Lib\site-packages

(or similar for the version of Python you are using). You will also need to add the full path to the folder:

C:\Python26\Lib\site-packages\vtk

to your PATH environment variable.
