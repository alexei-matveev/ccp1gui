Installation of the software packages for the CCP1GUI under Windows

Base Packages

The installation of the CCP1GUI on Windows has been made considerably easier thanks to the nice people at Enthought who have created a Python distribution that contains most of the software dependancies required by the CCP1GUI.

Their Python installation program can be downloaded from:

http://www.enthought.com/python/

IMPORTANT!: Please make sure that you use the latest (version 1.0.0 as of this writing) of Enthought Python, as versions from 0.9.2 up until 1.0.0 did not include Tk and so did not work with the CCP1GUI.

Save the installatation program to somewhere on your computer (e.g your Desktop) and then run it by double-clicking on it.

It is important to let Enthought install the python in the default location of C:\Python24 - otherwise you will need to edit the ccp1gui.bat file to point it at your Python installation or set a PYTHON environment variable to point at excutable.

When prompted to "Select Components" for the install "Python interpreter and Libraries" as well as "Tcl/Tk (Tkinter, IDLE, pydoc)" should be selected by default, but if for any reason they are not, make sure that they are selected.

Once you have installed Python, you will need to reboot.

At this point you have the base packages installed and so can download a bundle, so only read on if you are installing the additional packages individually.

Additional Packages

Numerical Python

If you've installed the Enthought package as described above, you already have Numeric installed. If you need to install it separately, please vist the Numerical Python website at sourceforge and click on the Download Numerical Python link. Then select the numpy package to download and you will be presented with a page that offers both installers for windows as well as source builds.

Scientific Python

If you've installed the Enthought package as described above, you already have Scientific Python installed. If you need to install it separately, please vist the Scientific Python website at http://starship.python.net/~hinsen/ScientificPython/. The main page offers both installers for windows as well as downloads of the Python source code.

Python MegaWidgets

Pmw is just pure Python code, and so doesn't need to be installed as such, just copied into a place where Python can find it. A link to a tarred and zipped archive is available on the Pmw webpage. You should be able to unpack the archive with a zip utility such as Winzip, and it will unpack to create a folder called Pmw.

To install Pmw, copy this folder into the Python<VERSION>\Lib\site-packages folder of your Python installation. For example if you are using Python 2.3 and it is installed on the default place on your C: drive, you would copy the Pmw folder to:

C:\Python2.3\Lib\site-packages
